# 💱 Currency Converter

A modern and responsive **Currency Converter** web application built using **HTML, CSS, and JavaScript**. It allows users to convert currencies in real time using live exchange rates from an external API while providing a clean user interface with dynamic country flags and an intuitive user experience.

## ⚙️ How It Works

1. Enter the amount you want to convert.
2. Select the source currency.
3. Select the target currency.
4. Click the **Get Exchange Rate** button.
5. View the converted amount instantly.
6. Use the **Swap** button to exchange the selected currencies with a single click.
